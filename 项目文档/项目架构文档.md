## 一、项目概述

AIFriends 是一个前后端分离的 Web 项目：
- **后端**：Django 6.0.5（运行在 `http://127.0.0.1:8000`）
- **前端**：Vue 3 + Vite + TailwindCSS 4 + daisyUI 5（运行在 `http://localhost:5173`）
- 前后端域名不同，已配置 CORS 跨域支持
- 用户认证：JWT（`djangorestframework-simplejwt`）
- 页面风格：小红书 Web 首页风格

### 4.8 REST Framework + JWT 认证配置
```python
from datetime import timedelta

REST_FRAMEWORK = {
    'DEFAULT_AUTHENTICATION_CLASSES': (
        'rest_framework_simplejwt.authentication.JWTAuthentication',
    ),
}

SIMPLE_JWT = {
    'ACCESS_TOKEN_LIFETIME': timedelta(hours=2),       # 访问令牌有效期 2 小时
    'REFRESH_TOKEN_LIFETIME': timedelta(days=7),       # 刷新令牌有效期 7 天

    'ROTATE_REFRESH_TOKENS': True,                     # 刷新时轮换刷新令牌
    'BLACKLIST_AFTER_ROTATION': True,                  # 旧令牌加入黑名单

    'AUTH_HEADER_TYPES': ('Bearer',),
}
```

## 七、URL 路由架构

### 7.1 后端路由（backend/urls.py）
```
backend/urls.py（项目根路由）
│
├── admin/                          → Django 管理后台
├── /（include web.urls）            → web 应用路由
│   ├── api/token/                  → JWT 获取 token
│   └── api/token/refresh/          → JWT 刷新 token
│
└──（DEBUG 时额外添加）
    ├── /assets/*                   → static/frontend/assets/
    └── /media/*                    → media/
```

### 7.2 前端路由（frontend/src/router/index.js）

| 路径 | 组件 | 说明 |
|------|------|------|
| `/` | App.vue（直接渲染） | 首页（Sidebar + Header + FeedWaterfall） |
| `/about` | AboutView | 关于页 |

> 注意：首页由 `App.vue` 直接渲染，不经过 `RouterView`。`AboutView` 为懒加载。
