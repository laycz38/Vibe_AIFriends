## 一、项目概述

AIFriends 是一个前后端分离的 Web 项目：
- **后端**：Django 6.0.5（运行在 `http://127.0.0.1:8000`）
- **前端**：Vue 3（运行在 `http://localhost:5173`）
- 前后端域名不同，已配置 CORS 跨域支持
- 用户认证：JWT（`djangorestframework-simplejwt`）

## 四、settings.py 完整配置说明

### 4.1 注册应用（INSTALLED_APPS）
```python
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'rest_framework',     # REST API

```

### 4.2 中间件（MIDDLEWARE）
CORS 中间件放在最顶部：
```python
MIDDLEWARE = [
    'corsheaders.middleware.CorsMiddleware',   # 必须尽量靠前
    'django.middleware.security.SecurityMiddleware',
    'django.contrib.sessions.middleware.SessionMiddleware',
    'django.middleware.common.CommonMiddleware',
    'django.middleware.csrf.CsrfViewMiddleware',
    'django.contrib.auth.middleware.AuthenticationMiddleware',
    'django.contrib.messages.middleware.MessageMiddleware',
    'django.middleware.clickjacking.XFrameOptionsMiddleware',
]
```

### 4.3 允许的主机
```python
ALLOWED_HOSTS = ['127.0.0.1', 'localhost']
```

### 4.4 时区设置
```python
TIME_ZONE = 'Asia/Shanghai'     # 中国时区
```

### 4.5 数据库
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}
```

### 4.6 静态文件与媒体文件
```python
STATIC_URL = 'static/'

STATICFILES_DIRS = [             # 开发阶段使用，生产阶段需注释掉
    BASE_DIR / 'static',
]

MEDIA_URL = 'http://127.0.0.1:8000/media/'
MEDIA_ROOT = BASE_DIR / 'media'
```

### 4.7 跨域配置（CORS）
```python
CORS_ALLOW_CREDENTIALS = True

CORS_ALLOWED_ORIGINS = [
    'http://localhost:5173',     # 允许 Vue3 前端跨域请求
]
```

### 4.8 REST Framework + JWT 认证配置
```python
REST_FRAMEWORK = {
    'DEFAULT_AUTHENTICATION_CLASSES': (
        'rest_framework_simplejwt.authentication.JWTAuthentication',
    ),
}

SIMPLE_JWT = {
    'ACCESS_TOKEN_LIFETIME': timedelta(hours=2),       # 访问令牌有效期 2 小时
    'REFRESH_TOKEN_LIFETIME': timedelta(days=7),       # 刷新令牌有效期 7 天

    'ROTATE_REFRESH_TOKENS': True,                     # 刷新时轮换刷新令牌
    'BLACKLIST_AFTER_ROTATION': True,                  # 旧令牌加入黑名单

    'AUTH_HEADER_TYPES': ('Bearer',),
}
```

## 五、API 接口列表

### 5.1 JWT 认证接口（web/urls.py）

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/token/` | 获取 JWT token（登录） |
| POST | `/api/token/refresh/` | 刷新 JWT token |

请求示例（获取 token）：
```json
POST /api/token/
{
    "username": "admin",
    "password": "12138"
}
```

响应：
```json
{
    "access": "eyJ...",
    "refresh": "eyJ..."
}
```

| 路径 | 映射目录 |

## 六、Vue 3 前端配置

### 6.1 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.x | 前端框架 |
| Vite | 8.0.13 | 构建工具 + 开发服务器 |
| Vue Router | 4.x | 前端路由 |
| Pinia | 2.x | 状态管理 |

### 6.2 开发服务器

```bash
cd frontend
npm run dev
```

启动后访问 `http://localhost:5173/`，支持热更新（HMR）。

### 6.3 打包配置（vite.config.js）

```javascript
import path from 'path'

export default defineConfig({
  // ...
  build: {
    outDir: path.resolve(__dirname, '../backend/static/frontend'),
    emptyOutDir: true,
  },
})
```

打包命令：
```bash
cd frontend
npm run build
```

打包产物将输出到 `backend/static/frontend/`，Django 通过 `/assets/` 路由提供给用户。

### 6.4 前端路由（src/router/index.js）

| 路径 | 组件 | 说明 |
|------|------|------|
| `/` | HomeView | 首页 |
| `/about` | AboutView | 关于页 |

### 6.5 环境变量配置（开发阶段）

为避免 CORS 问题，`vite.config.js` 通常还需配置代理。当前项目通过 Django 端 CORS 配置（`CORS_ALLOWED_ORIGINS`）来解决。
